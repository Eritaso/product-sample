# 初期設定
```bash
git clone https://github.com/Eritaso/product-sample.git product-sample
cd product-sample
make initial
```

# 開発時
## 起動
```
make up
```

## 終了
```
make down
```

## テスト実行
```
make test-unit
make test-feature
make test-integration
```
